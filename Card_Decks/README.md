## Card Type Descriptions

### Prompt Cards (18)
**Challenge:** Write effective AI prompts for business scenarios
**Time:** 5-8 minutes
**Skills:** Prompt engineering, context specification, constraint definition
**Example:** "Create a prompt for AI to generate product descriptions from specs"

### Debate Cards (18)
**Challenge:** Argue for/against AI applications and policies
**Time:** 8-12 minutes
**Skills:** Argumentation, ethical reasoning, persuasion
**Example:** "Debate whether AI-generated art deserves copyright protection"

### Prediction Cards (12) ⭐NEW
**Challenge:** Predict top 3 risks before seeing actual outcomes
**Time:** 8-10 minutes
**Skills:** Anticipatory thinking, risk identification, reasoning
**Special:** Hidden outcomes revealed after grading
**Example:** "Predict what could go wrong with AI resume screening"

### Risk Assessment Cards (12) ⭐NEW
**Challenge:** Identify risks across all 6 framework categories
**Time:** 12-15 minutes
**Skills:** Systematic analysis, framework application, severity rating
**Special:** Hidden answer keys for instructor reference
**Example:** "Assess risks in healthcare AI using cost as proxy for need"

## Important Notes for Printing

### What to Print for Students
✅ **Print:**
- Card title and number
- Scenario description
- Instructions/challenge
- Response format requirements
- Time allocation

❌ **DO NOT Print:**
- "EVALUATION CRITERIA (Hidden from students)" sections
- "What Actually Happened" sections (Prediction Cards)
- "Answer Key" sections (Risk Assessment Cards)
- Instructor grading notes

### Hidden Sections
Each card has sections at the bottom for instructor use only:
- **Prediction Cards:** Actual outcomes and scoring rubric
- **Risk Assessment Cards:** Complete answer key with all risks
- **All Cards:** Grading criteria and example responses

Keep these sections in your instructor reference materials only.

## Using the Card Decks

### For New Instructors
1. Start with **Prompt Cards 1-6** (easiest to grade)
2. Progress to **Prediction Cards 1-4** (simple scenarios)
3. Introduce **Debate Cards** (medium complexity)
4. Advance to **Risk Assessment Cards** (most complex)

### Card Selection Guide
- **Early Course:** Prompt Cards, simple Debates
- **Mid Course:** Predictions, complex Debates
- **Late Course:** Risk Assessments, synthesis
- **Final Exam:** Mix of all types

### Difficulty Progression
**Easiest → Hardest:**
1. Prompt Cards 1-6
2. Debate Cards 1-6, Prediction Cards 1-6
3. Prediction Cards 7-12, Risk Assessment Cards 1-6
4. Risk Assessment Cards 7-12

## File Formats

### Markdown (.md)
- Primary format for all cards
- Easy to read and edit
- Preserves formatting
- Can be converted to other formats

### Text (.txt)
- Plain text version available
- Maximum compatibility
- No formatting preservation
- Backup option

## Maintenance

### When to Update Cards
- Student feedback identifies issues
- Scenarios become outdated
- New AI developments occur
- Assessment needs change

### Version Control
- Document changes in comments
- Keep old versions in archive
- Note effective vs. problematic cards
- Share improvements with community

## Quick Reference

**Need cards for today's class?**
1. Choose appropriate difficulty level
2. Print scenario sections only
3. Keep answer keys at instructor station
4. Review rubric before class

**First time using prediction/risk assessment cards?**
1. Read corresponding documentation first
2. Review framework handout
3. Practice grading with rubric
4. Start with simpler scenarios

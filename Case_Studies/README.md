# Case Studies Overview

This folder contains in-depth case studies for AI in Business course integration.

## Case Study Collection

### Case 1: Liberty Regional Bank
**Topic:** AI Hiring Algorithm Bias
**Industry:** Financial Services / Human Resources
**Focus:** Algorithmic discrimination in recruitment

**Learning Objectives:**
- Understanding historical bias in training data
- Recognizing proxy variables for protected characteristics
- Evaluating fairness in AI hiring systems
- Considering regulatory and ethical implications

**Key Issues:**
- Gender bias in resume screening
- Credential evaluation disparities
- Impact on diversity and inclusion
- Legal compliance challenges

## Case Study Structure

Each case includes:
- **Background:** Company context and situation
- **The Decision:** AI system implementation details
- **Stakeholders:** Affected parties and perspectives
- **Outcomes:** What actually happened
- **Discussion Questions:** For class analysis
- **Supplementary Materials:** Articles, data, examples

## Classroom Use

### Discussion Format (60-75 minutes)
1. **Introduction** (5 min) - Present case context
2. **Individual Review** (10 min) - Students read case
3. **Small Group Discussion** (15 min) - Teams analyze
4. **Class Discussion** (20 min) - Facilitate debate
5. **Instructor Synthesis** (10 min) - Key takeaways
6. **Card Game Connection** (10 min) - Apply frameworks

### Written Analysis Assignment
- Individual or team case analysis paper
- Apply course frameworks
- Research similar real-world examples
- Propose recommendations

### Debate Format
- Assign stakeholder roles
- Teams prepare positions
- Formal debate structure
- Class votes on best arguments

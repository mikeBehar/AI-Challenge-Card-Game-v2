## Creating Your Own Templates

### Leaderboard Template Setup

**Recommended Structure:**
```
Team Name | Card 1 | Card 2 | Card 3 | Session Total | Running Total
Team 1    |   8    |   9    |   7    |     24        |      24
Team 2    |   10   |   7    |   8    |     25        |      25
[etc...]
```

**Tips:**
- Use conditional formatting for visual appeal
- Highlight top scores
- Track by card type for analysis
- Include date/session identifiers
- Make backup copies regularly

### Card Template Setup

**Required Sections:**
1. Card Header (type, number, difficulty)
2. Scenario Description
3. Challenge Statement
4. Response Format Instructions
5. Time Allocation
6. Hidden Answer Key
7. Grading Rubric Reference

**Optional Sections:**
- Related course concepts
- Prerequisite knowledge
- Extension activities
- Real-world examples
